# GitHub Actions

#### Short note about how to use GitHub actions for php projects 

## GitHub Actions Points
- OS
- Runners
- Workflow
    - Cancelling and Skipping Workflow runs
- Jobs
    - Parallel vs Sequential job execution 
    - Depended job execution
- Steps
- Events
    - Event filter 
- Triggers
- Environments
- Context 
- Conditions 
- Ignore Workflow 
    - for Readmea.md 
    - for any condition or branches
